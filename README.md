### Hey, I'm Hamza.

I build Python CLI tools and ship them on [PyPI](https://pypi.org/user/hamzaplojovic/) and [Homebrew](https://github.com/hamzaplojovic/homebrew-tap).

---

#### What I'm working on

**[remind-cli](https://github.com/hamzaplojovic/remind)** — A cross-platform CLI reminder engine with AI-powered suggestions and smart nudges.

```bash
# Install via pip
pip install remind-cli

# Or via Homebrew
brew tap hamzaplojovic/tap
brew install remind-cli
```
